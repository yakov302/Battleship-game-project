# Battleship-game-project

  
### Requirements
   - C++ compiler
   - linux (ubuntu) virtualization (GUI) enable.
   -  Installing SFML library - https://www.sfml-dev.org/tutorials/2.5/start-linux.php.
  
  
### Run the game   
   * go to directory - Battleship-game-project/src
   * run command:
```bash
   make check
```
